# Axelar Token Bridge App

Simple app to bridge tokens using a deposit address from Axelar.

## Stack

- RainbowKit
- Next
- Wagmi
- Axelar JS SDK
- Tailwind
- Framer Motion
- React Query

## Deployment

The Axelar Bridge application is deployed and accessible [here](https://axelar-bridge-app.vercel.app/).
